Setting Up

Use the array of tokens you obtained, either from GetTokens or GrabNewTokens, to create a new instance of the Friend class:

$friends = new \PSN\Friend($tokens);

You can now use $friends to make requests related to the logged in user's friends list.

MyFriends([$Filter = "online", $Limit = 36])

MyFriends takes zero arguments, but you can pass two optional arguments:

  • $Filter - used to filter which friends get returned. By default, this will grab online friends only.
  • $Limit - the amount of friends to return. 36 is the standard value used by Sony but you can adjust it.

$my_friends = $friends->MyFriends();
//you can use var_dump to see the contents of this object
var_dump($my_friends);

GetInfo($PSN)

GetInfo takes one argument, which is the name of the account you want to grab the info for. Note that this function might fail based on the user's privacy settings:

$my_friend = $friends->GetInfo("tustin25");
//you can use var_dump to see the contents of this object
var_dump($my_friend);

GetFriendsOfFriend($PSN [, $Limit = 36])

GetFriendsOfFriend requires one argument, which is the PSN you want to get the friends of. You can pass an optional second argument to limit how many friends get returned. Note that this function might fail based on the user's privacy settings:

$friends_of_my_friend = $friends->GetFriendsOfFriend("tustin25");
//you can use var_dump to see the contents of this object
var_dump($friends_of_my_friend);

Add($PSN [, $RequestMessage = ""])

Add requires one argument, which is the PSN you want to add to your friends list. You can pass an optional second argument to send a message along with your friend request:

$friends->Add("foobar", "add me plz!");

Remove($PSN)

Remove requres one argument, which is the PSN you want to remove from your friends list. Note that this user obviously must be on your friends list, otherwise it will fail:

$friends->Remove("foobar");
